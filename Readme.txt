Incorpora un mecanismo de búsqueda PBO en tu sitio utilizando un shortcode.

== Descripción ==

El plugin PBO Search te permite insertar un formulario de búsqueda de transfers con las preferencias del usuario en tu sitio web mediante el uso de un shortcode. Esto facilita que los visitantes de tu sitio realicen búsquedas personalizadas de transfers y obtengan resultados relevantes.

El shortcode para mostrar el formulario de búsqueda es: `[formulario_reservas]`.

== Instalación ==


1. Descarga y activa el plugin a través del menú 'Plugins' en WordPress.
2. Utiliza el shortcode `[formulario_reservas]` en tus páginas o entradas para mostrar el formulario de búsqueda PBO.

== Configuración ==

Para personalizar la configuración del plugin, ve al menú 'PBO tranfers' en el panel de administración de WordPress.
